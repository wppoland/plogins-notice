# Notice

Notice adds a dismissible, store-wide announcement bar to your WooCommerce site for sales, shipping notices and promos. Set a message, an optional call-to-action link and custom colours, and let shoppers dismiss it.

## Features

- Message with a small safe-HTML allow-list and an optional call-to-action link.
- Custom background, text and accent colours, pinned to the top of every page.
- Dismiss option remembered in the visitor's browser via localStorage — no cookies, no PII.
- Live preview, inline help and full input sanitisation in the settings screen.
- Assets load only when the bar is active, so it adds nothing to other pages.

## Installation

1. Upload the plugin to `/wp-content/plugins/notice`, or install it from Plugins → Add New.
2. Activate it. WooCommerce must be active.
3. Go to WooCommerce → Announcement Bar to write your message and set colours.

## Frequently Asked Questions

**Does it require WooCommerce?**
Yes.

**Does dismissing the bar use cookies?**
No. Dismissal is remembered in the visitor's browser with localStorage, so no cookies or personal data are involved.
